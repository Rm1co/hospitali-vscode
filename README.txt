Hospital Management System - Starter
-------------------------------------
Structure:
- index.html, patients.html, appointments.html, staff.html, inventory.html, billing.html, reports.html
- assets/css/style.css
- assets/js/*.js (frontend module code)
- backend/java/strath/hospital/... (Java skeleton)
- backend/database/schema.sql (example schema for MySQL)

How to use:
1. Download and extract this folder.
2. Open in VS Code. Use Live Server to view the frontend (open index.html).
3. To connect backend: create a MySQL database, update DatabaseConfig.java with credentials and build Java web app (Servlets / Spring etc.).
